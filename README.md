### Attract-Social ###
Attract Social is a social media app built using ReactJS, Redux Toolkit, React-Redux, Tailwind and Mockbee (mock backend)

Features Present:-
- Login
- Logout
- Signup
- Create a Post
- Like
- Comment and Replies
- Edit & Delete Post
- User Feed
- Sort by Date and Trending
- User Profile
- Bio
- Portfolio URL
- Follow/Unfollow
- Explore Feed
- Bookmark Post
- Loading

Experience the new society - Attract-Social Link
